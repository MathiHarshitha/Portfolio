## Harshitha Mathi Portfolio

Personal web developer portfolio built with Next.js and deployed on Vercel.

This project showcases real world production experience, frontend engineering skills, and live projects. The focus is on clean UI, performance, accessibility, and maintainable code.

---

## Tech Stack

- Next.js  
- React  
- TypeScript  
- Tailwind CSS  
- Vercel  

---

## Highlights

- Production grade portfolio with live deployment  
- Real world project experience, not demo apps  
- Clean component structure and modern frontend practices  
- Optimized for performance and basic SEO
